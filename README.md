# 老宅三叉路

![網站截圖](assets/images/screenshot.png)

## 專案簡介
「老宅三叉路」是一個整合政府老宅相關政策的資訊網站，幫助民眾一次看懂：
- **路徑A：修繕延壽** - 老宅延壽計畫補助
- **路徑B：原址改建** - 危老條例容積獎勵
- **路徑C：都更重建** - 都市更新稅賦優惠

## 目標用戶
- 👨‍👩‍👧 **幫長輩詢問的年輕人**：快速輸入基本資料，看懂「我家可以申請什麼補助？」
- 🏢 **顧問公司/從業人員**：查詢適用方案、法規條文、試算獎勵額度

## 技術架構
- 前端：HTML5, CSS3, JavaScript (原生，無框架)
- 部署：GitHub + Render (靜態網站)

## 資料來源
所有資訊皆來自政府公開資料：
- 內政部營建署
- 各縣市都市更新處
- 政府資料開放平臺

資料更新日期：2024年

## 免責聲明
本網站僅整理公開資訊，實際申請條件、補助金額以各主管機關最新公告為準。建議申請前向當地諮詢窗口確認。
